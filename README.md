## Installaiton steps

1. [Laravel Create project](https://laravel.com/docs/10.x/installation)
    - run `composer create-project laravel/laravel example-app`
2. Install Breeze
    - run `composer require laravel/breeze --dev `
    - run `php artisan breeze:install`
